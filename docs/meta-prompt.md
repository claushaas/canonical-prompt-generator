# META-PROMPT — GERADOR DE PROMPT CANÔNICO (FONTE ÚNICA EM ASSETS)

Este arquivo existe apenas como **ponte de documentação**.  
O conteúdo canônico do meta‑prompt está em:

- `assets/meta-prompt.md`

Motivo: a extensão lê o template diretamente de `assets/`, garantindo **single source of truth** entre documentação e geração do prompt.

Como editar:

1. Atualize **somente** `assets/meta-prompt.md`.
2. Caso precise consultar o conteúdo completo, abra o arquivo em `assets/`.
